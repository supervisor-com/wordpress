supervisor.com load testing and monitoring plugin for WordPress.

== Description ==

supervisor.com is a service to test and monitor your site with real web browsers. Once installed, supervisor.com will send virtual users who click, scroll and interact with your site from 1 to 100,000s. Be prepared for campaign traffic and black friday and know when the site is slow or not working.

supervisor.com WordPress plugin enables 1-click load tests for any WordPress site.

== Plugin Features ==

- Easy creation of a new supervisor.com monitoring and load tests
- Enables supervisor.com unlimited load tests
- Provides secure transmission of CPU usage statistics for supervisor.com
